# 🔔 yatte（やって）

**クリアマネージャー** - タスクが自動で消えていくシンプルなタスク管理アプリ

## コンセプト

「やりっぱなしを防ぐ」「溜まらないタスクリスト」をテーマにした軽量タスク管理アプリ。
タスクは自動削除されるため、常にリストがクリーンな状態を保てる。
