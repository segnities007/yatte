# コンセプト

## アプリ名

**yatte（やって）** - クリアマネージャー

## テーマ

「**やりっぱなしを防ぐ**」「**溜まらないタスクリスト**」

## 課題

従来のTODOアプリの問題点：

- タスクが溜まっていく一方
- 未完了タスクがストレスになる
- リストが肥大化して管理しきれない

## 解決策

**自動削除**でタスクが溜まらない仕組み：

- アラーム後24時間で自動削除
- 常にクリーンなリスト状態を維持
- 未完了のプレッシャーから解放

## 差別化ポイント

| ポイント | 説明 |
|---------|------|
| 自動クリーンアップ | タスクが溜まらない |
| シンプル | 必要最小限の機能 |
| ストレスフリー | 未完了タスクのプレッシャーがない |
| クロスプラットフォーム | どのデバイスでも同じ体験 |

## ターゲットユーザー

- シンプルなリマインダーが欲しい人
- TODOアプリで挫折した経験がある人
- 複数デバイスを使う人
