## カラーパレット

| 用途 | カラー | 説明 |
|------|--------|------|
| プライマリ | 🟢 **グリーン** | メインアクセント、ボタン・FAB |
| セカンダリ | 🟡 **クリームイエロー** | やわらかなアクセント |
| ライトモード背景 | ⚪ **ホワイト** | カード・背景 |
| ダークモード背景 | ⚫ **ブラック** | カード・背景 |

### カラーコード（参考）

```
// ライトモード
Primary: #4CAF50 (グリーン)
Secondary: #FFF8E1 (クリームイエロー)
Background: #FFFFFF
Surface: #F5F5F5

// ダークモード
Primary: #81C784 (ライトグリーン)
Secondary: #FFE082 (ゴールデンイエロー)
Background: #121212
Surface: #1E1E1E
```

## デザインキーワード

| キーワード | 説明 |
|-----------|------|
| ミニマル | 余計な要素を排除 |
| クリーンな余白 | 十分なパディング・マージン |
| やわらかな角丸 | 8dp〜16dp程度のラウンド |
| 軽やかなシャドウ | Elevation低め、柔らかいシャドウ |

---

## コンポーネントスタイル

### カード

- 角丸: 12dp〜16dp
- シャドウ: elevation 2dp〜4dp
- パディング: 16dp

### ボタン

- FAB: プライマリカラー、丸型
- テキストボタン: プライマリカラー
- 角丸: 8dp

### 入力フィールド

- スタイル: Outlined
- 角丸: 8dp
- フォーカス時: プライマリカラーのボーダー

### ボトムシート

- 角丸（上部のみ）: 16dp〜24dp
- ハンドル: 表示あり

## タイポグラフィ

|------|--------|---------|
| 見出し | 24sp | Bold |

## アニメーション

| 要素 | アニメーション |
|------|---------------|
| 画面遷移 | スライド / フェード |
| ボトムシート | スライドアップ |
| タスク完了 | スケール + フェードアウト |
| 削除 | スワイプ + フェードアウト |
