# Yatte デザイン原則

> **「楽しくタスク管理ができる」** — ゲーム感覚で継続できるタスクアプリ

> [!WARNING]
> 本ドキュメントに記載されているカラーコードや数値は **Design Systemの実装定義** です。
> Featureモジュール開発時は、これらの値を直接使用せず、必ず `YatteTheme` や `YatteColors` 経由で参照してください。

---

## 🎮 ブランド・パーソナリティ

### キーワード
- **楽しい** (Fun)
- **遊び心** (Playful)
- **継続できる** (Sustainable)
- **ゲーム感覚** (Game-like)

### インスピレーション
- **任天堂のUI/UX**: 心地よいフィードバック、ストレスフリーな操作感
- **ゲームデザイン**: 達成感、報酬、視覚的フィードバック

### 設計思想
| 原則 | 説明 |
|------|------|
| **Nintendo Quality** | すべての操作に心地よいフィードバック（bounceClick） |
| **Joy of Completion** | タスク完了が気持ちいい（コンフェティ、サウンド） |
| **Sustainable Fun** | 飽きずに継続できるゲーミフィケーション要素 |
| **Approachable** | 親しみやすく、プレッシャーを感じさせない |

## 🌿 カラーパレット (定義値)

実装時は `YatteTheme.colors.primary` のようにセマンティック名で参照します。

### プライマリカラー（グリーン系）

| トークン名 | Hex (Reference) | 用途 |
|------|-----|------|
| `preview` | `#2E7D32` | 強調、ヘッダー、重要アクション |
| `primary` | `#4CAF50` | メインアクション、成功状態 |
| `secondary` | `#81C784` | サブアクション、ホバー状態 |
| `tertiary` | `#C8E6C9` | 背景アクセント、選択状態 |

### アクセントカラー

| トークン名 | Hex (Reference) | 用途 |
|------|-----|------|
| `accentSunshine` | `#FFC107` | 注意、ハイライト、達成 |
| `accentHoney` | `#FFB300` | 警告、進行中 |
| `accentSky` | `#2196F3` | 情報、リンク |
| `accentCoral` | `#FF7043` | 通知、緊急 |
| `accentBerry` | `#AB47BC` | サウンド、特別 |

### ダークモード

| トークン名 | Hex (Reference) | 用途 |
|------|-----|------|
| `background` | `#0D1F0F` | 背景（最も暗い） |
| `surface` | `#1A2E1A` | Surface |
| `surfaceVariant` | `#2D3D2D` | カード背景 |
| `outline` | `#4A5D4A` | ボーダー、セパレーター |

> **Note**: ダークモードは「緑がかった森の中」をイメージ

### カラー使用ガイドライン

```
✅ アクセントカラーは大胆に使用
✅ 各カテゴリや状態に色で意味を持たせる
✅ 緑を基調としつつ、黄色で活気を出す

❌ 色を使いすぎて雑然とさせない
❌ 重要でない要素にアクセントカラーを使わない
```

## ✍️ タイポグラフィ

`YatteTypography` を使用します。

### フォントファミリー

| 用途 | フォント | 特徴 |
|------|----------|------|
| **見出し・本文** | Nunito | 丸みがあり柔らかい、フレンドリー |
| **日本語フォールバック** | Noto Sans JP | Nunitoと調和する丸ゴシック |
| **数字（スコア等）** | Nunito Bold | ゲーム的な強調 |

### タイプスケール

| スタイル | サイズ | 用途 |
|----------|--------|------|
| `displayLarge` | 36sp | ゲームオーバー的な大きな表示 |
| `headlineLarge` | 28sp | 画面タイトル |
| `headlineMedium` | 24sp | セクションタイトル |
| `titleLarge` | 22sp | カードタイトル |
| `titleMedium` | 18sp | リスト項目タイトル |
| `bodyLarge` | 16sp | 本文 |
| `bodyMedium` | 14sp | 補足テキスト |
| `labelSmall` | 12sp | ラベル、バッジ |

### 文字間隔（ゆったり）

```kotlin
// YatteTypography定義
letterSpacing = 0.5.sp  // 本文
letterSpacing = 1.sp    // 見出し
lineHeight = 1.5.em     // 本文の行間
```

## 🎬 モーション・アニメーション

`YatteMotion` を使用します。

### bounceClick (Nintendo Quality)

すべてのインタラクティブ要素に適用：

```kotlin
// YatteMotion.BounceClick 定義
stiffness = MediumLow
dampingRatio = MediumBouncy
```

### 画面遷移

| 遷移タイプ | アニメーション |
|------------|---------------|
| 前進（詳細へ） | 右からスライドイン |
| 後退（戻る） | 左へスライドアウト |
| モーダル | 下からスライドアップ |
| ダイアログ | スケール + フェード |

### フィードバックエフェクト

| イベント | エフェクト |
|----------|----------|
| タスク完了 | 🎊 コンフェティバースト |
| レベルアップ | ✨ シマー + スケールアップ |
| バッジ獲得 | 🏅 ゴールドグロー |
| ストリーク継続 | 🔥 パルスアニメーション |

## 🎮 ゲーミフィケーション要素

### 視覚的フィードバック

| 要素 | 実装 |
|------|------|
| **タスク完了** | コンフェティパーティクル |
| **連続達成** | 炎アイコン + カウンター |
| **新記録** | 特別なアニメーション |
| **レベル表示** | プログレスバー + レベル数 |

### ゲーミフィケーション機能（将来計画）

| 機能 | 説明 |
|------|------|
| **経験値 (XP)** | タスク完了で獲得、レベルアップ |
| **ストリーク** | 連続日数のカウント、ボーナス |
| **バッジ** | 特定の達成で獲得（早起き、完璧な週など） |
| **チャレンジ** | 週間/月間チャレンジ |
| **統計** | 可視化されたパフォーマンスデータ |

### 達成感の演出

```
タスク完了フロー:
1. タップ → bounceClick
2. チェックアニメーション
3. カード縮小 → フェードアウト
4. コンフェティバースト
5. (オプション) サウンドエフェクト
6. XP加算表示 「+10 XP」
```

## ✅ デザインチェックリスト

新しいUIを設計する際の確認項目：

### ブランド一貫性
- [ ] 任天堂的な心地よさを感じるか
- [ ] 遊び心があるか
- [ ] プレッシャーを感じさせないか

### カラー
- [ ] 緑ベースの配色になっているか
- [ ] アクセントカラーは意味を持って使われているか
- [ ] **YatteColorsトークンを使用しているか** (Raw Hex禁止)

### タイポグラフィ
- [ ] **YatteTypographyを使用しているか**
- [ ] 余白は十分にあるか（ゆったり）

### インタラクション
- [ ] すべてのボタン/タップ可能要素にbounceClickがあるか
- [ ] フィードバックが即座に分かるか

### ゲーム要素
- [ ] 達成感を演出できているか
- [ ] 継続したくなる仕組みがあるか
