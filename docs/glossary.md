# 用語集 (Glossary)

yatte プロジェクトで使用される専門用語と定義。

## ドメイン用語

| 用語 | 英語 | 説明 |
|------|------|------|
| **スタートリスト** | Start List | アプリのメイン画面。今日やるべきタスクの一覧。 |
| **1回きりタスク** | One-time Task | 一度完了するか、24時間経過すると消えるタスク。 |
| **週ループタスク** | Weekly Loop Task | 毎週決まった曜日に復活するタスク。自動削除されない。 |
| **n分前通知** | Minutes Before | アラーム予定時刻の何分前に通知を送るかの設定。 |
| **自動削除** | Auto Cleanup | アラーム発火後24時間経過したタスクをリストから消去する機能。 |
| **タイムライン** | Timeline | 完了したタスクの履歴を時系列で表示する画面。 |
| **スキップ** | Skip | 週ループタスクを、今回（今週）だけ非表示にする機能。 |

## 技術用語

| 用語 | 説明 |
|------|------|
| **Compose Multiplatform** | Kotlinで記述するクロスプラットフォームUIフレームワーク。Android/iOS/Desktopに対応。 |
| **MVI** | Model-View-Intent。単方向データフローを採用したUIアーキテクチャパターン。 |
| **Aggregate** | 集約。ドメイン駆動設計(DDD)における、整合性を保つデータのまとまり。 |
| **UseCase** | ビジネスロジックの単位。1つのユーザー操作に対応することが多い。 |
| **Repository** | データの保存・取得を抽象化する層。Domain層で定義し、Data層で実装する。 |
| **Room** | Android/KMP向けのSQLiteオブジェクトマッピングライブラリ。 |
| **Koin** | Kotlin向けの軽量な依存性注入(DI)フレームワーク。 |
| **FNB** | Fixed Navigation Bar（ボトムナビゲーションバー）。画面下部の固定メニュー。 |
