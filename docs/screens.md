# 画面仕様

## ナビゲーション

| 項目 | 内容 |
|------|------|
| 方式 | **ドロワー**（左からスライドで出るメニュー） |
| トリガー | ハンバーガーメニュー or スワイプ |

---

## 画面一覧

| 画面 | 説明 | 主な要素 |
|------|------|---------|
| ホーム | メイン画面 | 今日のタスク一覧、FAB |
| スタートリスト | タスク管理 | 全タスク一覧、フィルター |
| 履歴 | タイムライン表示 | 線形グラフ、日付選択 |
| 設定 | 各種設定 | 設定項目一覧 |

---

## タスク追加フロー

| 項目 | 内容 |
|------|------|
| トリガー | **FAB**（右下の＋ボタン） |
| UI | **ボトムシート**（下から出るシート） |

### ボトムシート内の入力項目

1. タイトル（テキスト入力）
2. 時刻（タイムピッカー）
3. n分前（数値入力 or スライダー）
4. 週ループトグル
5. 曜日選択（週ループON時のみ表示）

---

## 設定画面

### 設定項目

| 項目 | 説明 | UI |
|------|------|-----|
| デフォルトn分前 | アラームのデフォルト値 | 数値入力 |
| 通知方法 | 音/バイブ/プッシュ | チェックボックス |
| 通知音 | カスタム音源管理 | リスト + 追加ボタン |
| テーマ | 外観設定 | ラジオボタン |
| データリセット | 全削除 | ボタン + 確認ダイアログ |

### テーマ設定

| 選択肢 | 説明 |
|--------|------|
| ライト | 白ベース |
| ダーク | 黒ベース |
| システム連動 | OSの設定に従う |
