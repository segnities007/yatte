## 1. ホーム画面 (Home)

アプリのメイン画面。今日やるべきことに集中できる設計。

### レイアウト仕様

- **ヘッダー**:
    - 中央: 現在表示中の日付（例: "12月25日(水)"）
    - タップアクション: 「今日」に戻る
    - 背景: トップバー背景色
- **ボディ**:
    - `HorizontalPager` による日付スワイプ切替
    - **空の状態**: "No tasks for this day" イラストとメッセージ
    - **タスクあり**: 縦スクロールリスト
- **FAB (Floating Action Button)**:
    - 位置: 右下
    - アイコン: `Add` (+)
    - アクション: [タスク作成画面](#5-タスク作成画面-taskform)へ遷移

### インタラクション

- **タスク完了**: チェックボックスをタップ → `COMPLETED` アニメーション
- **リスト更新**: 日付移動時に該当日のタスクを非同期ロード
- **Pull to Refresh**: なし（ローカルDB監視で自動更新）

## 2. タスク管理画面 (Management)

登録済みの全タスクを俯瞰・管理する画面。

### レイアウト仕様

- **リスト形式**:
    - **週ループタスク**: 曜日情報・スキップ状態を表示
    - **1回きりタスク**: 未来の予約タスクを表示
- **カード情報**:
    - タイトル、時間、繰り返し設定
    - スイッチ: 一時的な有効/無効切り替え（将来）
    - メニュー: 編集 / 削除 / [今週スキップ]
    - **検索**: ヘッダーの検索アイコンからフィルタリング可能

### アクション

- **編集**: タップで編集モードへ
- **削除**: スワイプまたはメニューから削除（確認ダイアログあり）
- **スキップ**: 週ループタスクに対して実行可能

---

## 3. 履歴画面 (History)

### レイアウト仕様

- **ヘッダー**:
    - 完了タスク総数 / 週間達成率
- **タイムライン**:
    - 縦軸: 日付（降順）
    - コンテンツ: その日に完了したタスクのタイトルと時間
    - **検索**: ヘッダーの検索アイコンから過去ログを検索可能
- **グラフ**:
    - 過去7日間の完了数を棒グラフ表示（将来）

## 5. タスク作成画面 (TaskForm)

ホームのFABから遷移するフルスクリーンダイアログ。

### 入力フィールド構成

1. **タイトル** (TextField)
    - プレースホルダー: "何をする？"
    - オートフォーカス: ON
2. **時刻設定** (TimePicker)
    - アナログ文字盤 (Clock Dial) または 数値入力
    - キーボード/タッチモード切替ボタン
3. **通知タイミング** (Slider)
    - ラベル: "n分前"
    - 範囲: 0〜60分
4. **繰り返し設定** (Chip / Toggle)
    - 選択: 1回のみ / 毎週
5. **曜日選択** (Multi-select Chips)
    - `毎週`選択時のみ表示
    - 月〜日を複数選択可

### アクションバー

- **キャンセル**: 左上 `×` ボタン
- **保存**: 右上 `保存` ボタン（タイトル入力時のみ有効）
