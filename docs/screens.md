# 画面仕様

## ナビゲーション

| 項目 | 内容 |
|------|------|
| 方式 | **ボトムナビゲーションバー (FNB)** |
| トリガー | 画面下部のアイコン |

---

## 画面一覧

| 画面 | 説明 | 主な要素 |
|------|------|---------|
| ホーム | メイン画面 | 日別タスク一覧、日付スワイプ、FAB |
| タスク管理 | **[NEW]** 登録タスク一覧 | 登録済みタスク一覧、編集・削除 |
| カレンダー | 月間表示（将来） | カレンダー、タスク件数バッジ |
| 履歴 | タイムライン表示 | 線形グラフ、日付選択 |
| 設定 | 各種設定 | 設定項目一覧 |

---

## ホーム画面詳細

### 日付切り替え機能

| 項目 | 内容 |
|------|------|
| デフォルト表示 | **今日のタスク** |
| 日付移動 | 左右スワイプで前日/翌日へ移動 |
| 日付表示 | ヘッダーに「12月25日（水）」形式で表示 |
| 今日に戻る | ヘッダータップで今日に戻る |

### カレンダービュー（将来実装）

| 項目 | 内容 |
|------|------|
| 表示形式 | 月間カレンダー |
| タスク件数 | 各日にバッジで件数表示 |
| 日付タップ | タップした日のタスク一覧へ遷移 |
| 優先度 | Phase 2以降で実装予定 |

---

## タスク追加フロー

| 項目 | 内容 |
|------|------|
| トリガー | **FAB**（右下の＋ボタン） |
| UI | **フルスクリーン画面**（専用の入力画面） |

### 入力画面の項目

1. タイトル（テキスト入力）
2. 時刻（時・分入力）
3. n分前（スライダー）
4. タスクタイプ（1回きり/週ループ）
5. 曜日選択（週ループ時のみ表示）

---

## 設定画面

### 設定項目

| 項目 | 説明 | UI | ステータス |
|------|------|-----|--------|
| デフォルトn分前 | アラームのデフォルト値 | スライダー | ✅ |
| 通知音 | ON/OFF | スイッチ | ✅ |
| バイブレーション | ON/OFF | スイッチ | ✅ |
| カスタム通知音 | カスタム音源管理 | リスト + 追加ボタン | ✅ |
| テーマ | 外観設定 | セグメントボタン | ✅ |
| データリセット | 全削除 | ボタン + 確認ダイアログ | ✅ |

### テーマ設定

| 選択肢 | 説明 |
|--------|------|
| ライト | 白ベース |
| ダーク | 黒ベース |
| システム連動 | OSの設定に従う |
